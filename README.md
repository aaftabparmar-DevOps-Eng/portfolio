# Aaftab Parmar — DevOps & Cloud Engineer Portfolio

A modern, dark-themed personal portfolio for **Aaftab Parmar** — BCA 3rd Year student at Vidhyadeep University. Passionate about DevOps, Cloud Engineering, and Automation. Focused on building scalable, cloud-native infrastructure with modern DevOps practices.

Built with **React + TypeScript + Vite + Tailwind CSS + Framer Motion**. Designed for one-click deployment on **Vercel**.

## Stack

- React 18 / TypeScript
- Vite (build tool)
- Tailwind CSS (utility-first styling)
- Framer Motion (animations + scroll effects)
- Lucide React (icons)
- Kanit font (Google Fonts, weights 300-900)

## Sections

1. **Hero** — Animated name with gradient, tagline, DevOps introduction, and 3D image with blur background
2. **About** — Professional bio with glass morphism cards, education at Vidhyadeep University, floating DevOps icons
3. **Skills** — 10 categorized skill cards covering Containerization, Orchestration, Cloud, IaC, CI/CD, GitOps, Monitoring, Networking, Automation & Version Control
4. **Projects** — Sticky-stacking 3D cards for CapitalX, Wanderlust, Online Shopping App & AI-Powered Docker with GitHub links
5. **Contact** — Email, Phone, LinkedIn, GitHub with glass morphism cards & location badge (Surat, Gujarat)
